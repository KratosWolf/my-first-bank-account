# Brand — MyFirstBA2

## Identidade

- Nome: My First Bank Account
- Tagline: Educação financeira real para crianças
- Público principal: pais que querem ensinar finanças aos filhos (6-15 anos)

## Paleta de Cores

- Verde escuro (fundo): #0D2818
- Verde médio (cards): #1A4731
- Dourado (CTAs): #F5B731
- Amarelo claro: #FFD966
- Branco: #FFFFFF

## Tom de voz

- Para pais: profissional, confiável, como um app bancário real
- Para crianças: lúdico, encorajador, celebrativo (mascote porquinho)

## Referências visuais

- ArobixBank (visual para pais)
- App com mascote porquinho, micro-interações, animações
